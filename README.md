# 📊 Employee Sales Performance Dashboard (Excel)

## 📌 Project Overview

This project is an **interactive Sales Performance Dashboard built using Microsoft Excel**.
The dashboard analyzes the performance of sales executives across multiple cities and compares their **actual sales with assigned targets**.

The purpose of this project is to demonstrate **data analysis, pivot tables, data visualization, and basic VBA automation** skills using Excel.

---

## 🗂 Dataset

The dataset is included inside the Excel workbook in the **Raw_Data** sheet.

It contains sales data of employees working in different regions.

### Columns in the Dataset

* Employee Code
* Sales Executive
* Region / City
* Day 1 Sales
* Day 2 Sales
* Day 3 Sales
* Day 4 Sales
* Day 5 Sales
* Total Sales
* Target
* Target Hit %
* Away From Target %

---

## ⚙️ Project Workflow

### 1️⃣ Data Preparation

* Structured the raw dataset inside Excel
* Organized columns for analysis
* Calculated **Total Sales**
* Calculated **Target Hit %**
* Calculated **Away From Target %**

---

### 2️⃣ Data Analysis using Pivot Tables

Created multiple pivot tables to analyze:

* Total sales by sales executive
* Target achievement percentage
* Distance from sales target
* Region-wise employee performance

---

### 3️⃣ Dashboard Visualization

Built an interactive dashboard using different chart types:

* **Bar Chart** → Employee total sales comparison
* **Pie Chart** → Sales distribution
* **Line Chart** → Performance trend
* **Pivot Table summaries** for detailed insights

---

### 4️⃣ Interactive Controls

Added interactive elements to improve usability:

* City filters (Chennai, Delhi, Mumbai, Nagpur, Ranchi)
* Dashboard selection using **checkbox controls**
* Dynamic pivot table updates

---

### 5️⃣ VBA Automation

Used **VBA with Developer tools** to connect checkboxes with pivot tables so that users can manually control which dashboard section updates with filters.
